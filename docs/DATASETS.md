# Datasets

Medical image data and labels are intentionally excluded from Git. Keep them local unless you have explicit permission to publish them.

## Classification dataset

Expected structure:

```text
datasets/classification/
├── icon.yaml
├── Train/
│   ├── Glioma/
│   ├── Meningioma/
│   ├── No Tumor/
│   └── Pituitary/
└── Val/
    ├── Glioma/
    ├── Meningioma/
    ├── No Tumor/
    └── Pituitary/
```

## Segmentation dataset

Expected structure:

```text
datasets/segmentation/
├── data.yaml
├── images/
│   ├── train/
│   └── val/
└── labels/
    ├── train/
    └── val/
```

`data.yaml` uses YOLO segmentation label format. Keep image and label filenames aligned.

## Privacy checklist

- Remove patient names, IDs, hospital metadata, and acquisition metadata before sharing.
- Do not commit `.cache` files generated by training.
- Confirm the dataset license allows public redistribution before publishing any sample data.
